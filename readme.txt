This plugin help you improve SEO of your site by easing bulk alt changing  

== Description ==


NextGen magic seo image help you change images alt so quickly and easily .
in plugin admin page all posts tag's are listed.also you can add you'r own alts .there is an autocomplete mechanisem help you choose the best keywords as alt by using Google Suggest system.
there is two choices,you can set all selected images alts equally or set alts random,in bulk alt mode all keywords are concatenate to each other and assign to selected images.
in random mode keyword will assign to an image complettly random.


== Installation ==

1. Upload `nextgen-magic-seo-image` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
